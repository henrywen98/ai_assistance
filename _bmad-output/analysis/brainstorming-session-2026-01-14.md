### 快速捕获流程设计

```
触发 → 入口 → 输入 → 确认 → 反馈
  │      │      │      │      │
  │      │      │      │      └─ ✅ + 分类结果 + 动效音效

```

### 关键设计原则

- **捕获与处理分离**：碎片时间捕获，专注时间处理
- **AI 分类 + 批量审核**：信任 AI，但保留集中复核机制
- **适应不规律节奏**：不强迫固定习惯，利用随机空闲时刻
- **Mac 原生交互**：快捷键、Dock、菜单栏多入口

## PRD 素材整理

### 产品定位

**产品名称**：待定（AI 个人助理 / ADHD GTD 助手）

**一句话描述**：为 ADHD 用户设计的 AI 驱动个人助理，通过"零摩擦捕获 + AI 自动整理"帮助用户管理时间和事务。

**核心价值主张**：
- 🚀 **快速卸载**：想到什么，立刻丢进去，一身轻松
- 🤖 **AI 代劳**：分类、设时间、归档——所有细心操作都交给 AI
- 🧠 **ADHD 友好**：为工作记忆短、时间盲、没耐心的大脑设计

### 目标用户

**主要用户**：ADHD 人群 / 工作记忆短 / 长时间电脑工作者

**用户特征**：
- 灵感多但转瞬即逝
- 讨厌需要"细心"的操作
- 没有固定的时间管理习惯
- 时间感知弱（时间盲）
- 没耐心配置复杂功能

### 核心功能模块

#### 1. 快速捕获组件（MVP 核心）

```
触发 → 入口 → 输入 → 确认 → 反馈
  │      │      │      │      │
  │      │      │      │      └─ ✅ + 分类结果 + 动效音效
  │      │      │      └─ 按回车
  │      │      └─ Cmd+V 粘贴（截图/文本）
  │      └─ 快捷键 / Dock / 菜单栏
  └─ 看到信息，想记下来
```

**关键设计**：
- 空白输入框（视觉极简）
- 支持截图 + 文本粘贴
- 回车确认发送
- 反馈：已收到 + AI分类结果 + 动效音效

#### 2. AI 解析引擎

**输入**：自然语言（中文友好）
**输出**：
- 时间（如有）→ 日历
- 内容 notes → 任务/笔记
- 关联（可选）→ 尽力而为

**能力边界**：
- 自动分类：日历 / Todo / 笔记
- 自动设置时间
- 自动推荐优先级（重要/普通）
- 自动归档过期内容
- 智能关联相关灵感

#### 3. 三大容器（独立但关联）

| 容器 | 用途 | 核心属性 |
|------|------|----------|
| 📅 日历 | 时间承诺，"到点大概率发生" | 时间锚点，弥补时间盲 |
| ☑️ Todo | 提醒备忘，"记得有这个事" | 待处理池 |
| 📝 笔记 | 灵感保存，"未来可能用上" | 创意仓库 |

**跨容器操作**：
- 灵感 → 一键变任务
- 任务 → 一键加日历
- 日历 → 自动生成 Time Sheet

**信息流转**：
```
灵感(模糊) → 项目 → 子任务 → 日历(确定)
```

#### 4. Memory 记忆系统

存储用户的个性化信息，让 AI 越用越懂你：
- 个人偏好（习惯、工作时间）
- 分类规则（关键词→项目映射）
- 人际关系（谁是谁）
- 常用信息（地址、常用项目）

#### 5. 视图与展示

**日历三维视图**：
| 时间 | 功能 | 心理作用 |
|------|------|----------|
| 过去 | Time Sheet 记录 | 成就感 |
| 现在 | "我在哪个阶段" | 时间定位 |
| 未来 | 下一个时间块 | 安全感 |

**成就可视化**：
- GitHub 贡献图风格的色块
- 简单统计图表

#### 6. 批量处理模式

**设计理念**：捕获用碎片时间，处理用专注时间

- 用户主动进入（摸鱼时、空闲时）
- 批量审核 AI 分类结果
- 集中整理灵感

### 设计原则

| 原则 | 说明 |
|------|------|
| **零摩擦** | 从想法到记录，步骤越少越好 |
| **零决策** | 输入时不需要做任何分类决策 |
| **零配置** | 不需要耐心配置，开箱即用 |
| **信任优先** | AI 先做，用户事后纠错 |
| **尽力而为** | AI 能做多少算多少，不追求完美 |
| **适应用户** | 软件适应不规律的用户节奏 |

### MVP 范围

**✅ MVP 包含**：
- 快速捕获组件（Mac 端）
- AI 解析（分类、时间、优先级）
- 三大容器（日历、Todo、笔记）
- 基础 Memory 系统
- 成就可视化（GitHub 贡献图）
- 早上今日概览

**❌ MVP 不包含**：
- 标签系统
- 复杂统计报表
- 频繁提醒通知
- 分享/协作功能
- 主题定制
- 复杂重复任务规则
- 多端同步（未来）
- 语音输入（未来）

### 技术考量

**平台**：macOS（优先）
**入口方式**：
- 全局快捷键
- Dock 栏图标
- 菜单栏图标

**AI 能力**：
- 自然语言理解（中文）
- 内容分类
- 时间解析
- 关联推荐
